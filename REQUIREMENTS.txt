================================================================================
HAVEN — Requirements & Installation Guide
================================================================================

SYSTEM REQUIREMENTS
-------------------
  OS      : Windows 10 / 11 (64-bit)
  RAM     : 4 GB minimum, 8 GB recommended
  Disk    : 500 MB free space
  Display : 1280 x 720 minimum

RUNTIME REQUIREMENTS (development)
-----------------------------------
  Node.js : 18.x or later   https://nodejs.org
  npm     : 9.x or later    (bundled with Node.js)
  Git     : any recent      https://git-scm.com  (optional, for cloning)

  Note: Electron (~41.x) is downloaded automatically by npm install.
  It requires no separate installation.

================================================================================
INSTALLATION — DEVELOPMENT / SOURCE
================================================================================

1. GET THE SOURCE
   --------------
   Clone or download the repository to a local folder:

     git clone <repo-url>
     cd individualitymodel

   Or extract the ZIP and open a terminal in the project folder.

2. INSTALL DEPENDENCIES
   ---------------------
     npm install

   This downloads all packages listed in package.json including Electron,
   React, Vite, TypeScript, and electron-builder (~300–500 MB first run).

3. RUN IN DEVELOPMENT MODE
   ------------------------
   Opens a Vite dev server AND an Electron window with hot reload:

     npm run electron:dev

   The app window will appear automatically once the dev server is ready
   (usually within 10–15 seconds).

4. RUN WEB PREVIEW ONLY (no Electron)
   ------------------------------------
     npm run dev
   Then open http://localhost:5173 in a browser.

================================================================================
BUILD — PRODUCTION INSTALLER
================================================================================

1. BUILD THE DISTRIBUTABLE
   ------------------------
     npm run electron:build

   Output: release/  folder containing a Windows NSIS installer (.exe).

2. INSTALL ON WINDOWS
   -------------------
   Run the generated .exe in release/.
   The installer allows choosing the installation directory.
   A desktop shortcut and Start Menu entry are created automatically.

================================================================================
SCRIPTS REFERENCE
================================================================================

  npm run dev              Vite dev server only (browser)
  npm run build            TypeScript compile + Vite production build
  npm run preview          Preview production build in browser
  npm run electron:dev     Dev server + Electron window (hot reload)
  npm run electron:build   Full production build + Windows installer
  npm run electron:start   Start Electron against already-built dist/

================================================================================
TROUBLESHOOTING
================================================================================

  "electron not found" after npm install
      Run: npm install --save-dev electron
      Or delete node_modules/ and package-lock.json, then re-run npm install.

  App window does not open in electron:dev
      Vite may still be starting. Wait ~15 seconds. Check terminal for errors.

  Port 5173 already in use
      Kill the process on that port or change the port in vite.config.ts.

  Windows Defender / SmartScreen blocks the installer
      Expected for unsigned builds. Click "More info" → "Run anyway".
      For production release, sign the executable via electron-builder's
      certificateFile option in package.json.
